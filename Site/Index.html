Bienvenu

Quel est le but de ce site ?

Ce site a pour but de vous aider dans la Conception Orientée Objet (COO) en vous donnant les notions importantes sur le Langage de Modélisation Unifié (UML) qui donne une méthode pour visualiser la conception d'un système. Il vous fera partager différentes ressources pour améliorer votre niveau en conception orientée objet. Il est possible de s'évaluer pour connaître son niveau dans la rubrique QCM (il est conseillé de suivre tous les TD avant de faire les QCM). Les différents TP vous donneront les connaissances nécessaires pour le langage UML de façon structuré s'inspirant du site de MBF.Le langage UML permet également de générer le code permettant la programmation orientée objet.

Culture générale

Le langage UML s'est inspiré sur plusieurs méthodes notamment les méthodes BOOCH,OMT et OOSE qui sont les pères d'UML.
