Des modèles aux codes

Ce TD a pour but de vous apprendre à coder en ayant la conception.

Pour coder il faut se répartir les tâches et pas n'importe comment car plusieurs classes sont liés, donc les sous codes dépendront beaucoup des autres sous code. Il faut faire des packages et chacun code un package il y aura des relations qu'on ne pourra pas coder seul mais ces relations seront le moins nombreuses qu'on puisse faire.

Vous avez la possibilité d'obtenir le code avec modélio.
